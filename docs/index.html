Unzip

Unzip is a JavaFX application that displays the entries of a zip file and extracts selected entries. Entries can be filtered, and an entry can be viewed or edited by an external program by running a command that is associated with a set of filename extensions.

This is a preliminary document; more information on this project will be along later.
